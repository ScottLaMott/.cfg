# Dotfile-Verzeichns

## Plan
* mit bare-Repository
* mit dotbare-Plugin

## Install

  * Install
```
    git clone https://github.com/kazhala/dotbare.git $HOME/.oh-my-zsh/custom/plugins/dotbare
```

  * Plugin in .zshrc laden
```
    plugins=(... dotbare ... ...)
```
